SIPB HWOPS

Welcome to the website of SIPB's Hardware Operations (HWOPS) project!

Overview:

  • We are responsible for the operation and maintenance of the SIPB machine room in W20-575A, as well as for some aspects of SIPB's servers in W91.
  • In addition, we are responsible for planning out the future of SIPB's physical server spaces.
  • We provide physical server space, server hardware, and server support services to other SIPB projects.
  • If your project is not an official SIPB project, we will be able to refer you to other resources, but we will not be able to provide you with server space unless your project becomes recognized as an official SIPB project.
  • Some of our operational databases are publicly accessible on this website, for the sake of transparency.
  • As with all SIPB services, HWOPS is a volunteer-run service, and although we strive for perfect uptime of the server room, we will not always be able to provide it.
